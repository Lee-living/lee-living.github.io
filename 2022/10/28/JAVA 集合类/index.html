JAVA 集合类

Java集合类：Java集合类是可变大小链式存储的，也叫容器类；占用空间比数组大，但存储类型更丰富，可以是基本数据类型也可以存储对象，长度相比数组也可变.

集合类又主要由两个接口组成：Collection 和 map；

Collection

主要由三部分组成：Set、Queue、List

Set

Set是Java中的一种无序的，不能重复的集合类；Set接口有两个比较常用的实现，分别是HashSet和TreeSet

HashSet

主要特点是快速查找元素；HashSet是基于hash算法实现的，每次添加新对象都会根据散列码[^1]判断对象是否重复，所以才能保证HashSet是不重复的。

TreeSet

会自动进行自然排序（默认升序排列），和HashSet一样也不能重复；多数情况下可能需要自定义排序，可以定制排序，通过Comparator接口，重写compare(T o1,T o2)方法实现

Queue

队列，最大的特点就是先进先出，所有操作都是在固定位置进行，删除在队首进行，插入在队尾进行

List

List是一种允许重复元素的有序集合

主要有ArrayList、Vector、LinkedList三个实现

Arraylist

ArrayList是通过数组方法实现的，但长度可变

查询效率高，增删效率低

是非线性安全[^2]的

Vector

Vector 和 ArrayList 是相似的，但是是线程安全的

Vector类对集合的元素操作时都加了synchronized[^3]来保证线程安全

因为Vector线程安全，查询效率不如ArrayList

LinkList

LinkList是基于双向链表的，这点和Vector、ArrayList不同，也是因为基于双向链表，查询不如Vector 和 ArrayList，但增删更有优势

LinkList也是线程不安全的

Map

Map中比较常见的是 HashMap、TreeMap、Hashtable 类。

HashMap

HashMap是基于散列表的map接口的实现类，线程不安全

TreeMap

TreeMap是根据红黑树算法实现的，TreeMap最大的特性就是支持自然排序

红黑树（Red Black Tree）：是一种自平衡二叉查找树，时间复杂度低O(logN)，增删查改系哦啊效率都较高

HashTable

HashTable和 HashMap 基本一致，最大的区别是 HashTable 是线程安全的，HashMap 不是

[^1]: 散列码：通过object（）中的hash code()来实现的，散列码可以获取对象中的信息,然后将其转换成那个对象“相对唯一”的整数(int)，就可以通过散列码来进行查询，所以查询速度更快
[^2]: 线性安全：多线程同时访问同一个对象时，会不会产生数据污染，如果多线程工作时各个线程都可以正常的工作、获取倒正确的结果，那么它就是线程安全的
[^3]: Synchronized：能够保证在同一时刻最多只有一个线程执行该段代码
